**OSaF** (Open Source Activity Formatter) is a global UI macro for ServiceNow that renders updates, comments, work notes, and related records for a single record on UI16.

The out-of-box activity formatter had limitations that ServiceNow wasn't opening up, so I built this as an alternative.

## Download

[Download OSaF v0.1 Update Set XML](/assets/images/osaf%20v0.1.xml)

## Installing

1. Download the XML above
2. Install it as an update set in your instance
3. The UI macro is added to your system — it does **not** change any forms automatically
